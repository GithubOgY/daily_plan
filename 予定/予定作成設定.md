## 設定内容

### Google Maps MCP設定
- **使用**: 有効
- **移動時間計算**: Google Maps MCPを使用して自動計算
- **デフォルト移動手段**: 車（driving）
- **ラッシュアワー考慮**: 30分のバッファを追加

### 使用するGoogle Maps API
以下のAPIを利用します：

1. **Places API (New)**
   - 場所の検索に使用
   - バリアフリー対応施設の検索など

2. **Geocoding API**
   - 住所変換などに必要
   - 住所から座標への変換、座標から住所への変換

3. **Directions API**
   - 経路検索に必要
   - 出発地から目的地までの詳細な経路情報を取得

4. **Distance Matrix API**
   - 距離計算に必要
   - 複数の出発地・目的地の距離・時間を一度に計算

### 燃料代計算設定
- **有効**: 有効
- **燃費**: 1kmあたり15円
- **計算方法**: 移動距離（km）× 15円 = 燃料代
- **表示項目**: 
  - 移動距離（往路・復路・往復合計）
  - 燃費（1kmあたりの金額）
  - 燃料代（往路・復路・往復合計）

## 使用方法

1. 予定を作成する際、この設定ファイルを参照
2. 移動時間が必要な場合は、Google Maps MCPを使用して自動計算
3. 登録されている場所名を使用すると、住所が自動的に参照されます

---

## 予定作成テンプレート構造

予定ファイルには以下のセクションが含まれます：

1. **基本情報**: 日付、主要な場所など
2. **今日のタスク**: チェックリスト形式のタスク
3. **スケジュール**: 時刻別の予定表
4. **詳細メモ**: 移動時間、ルート、その他の詳細情報
5. **注意事項**: 重要な注意点や備考

---

## 移動時間計算について

- Google Maps MCPを使用して、出発地と目的地の間の移動時間を自動計算
- ラッシュアワー時間帯（17:00-19:00）は、通常の所要時間に30分のバッファを追加
- 計算結果は予定表の「詳細メモ」セクションに自動記録されます
