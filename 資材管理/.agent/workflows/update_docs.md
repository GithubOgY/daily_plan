description: コードの変更に基づいてドキュメントを更新する
---
このワークフローは、Code.gs や Dialog.html の変更内容を分析し、ドキュメント（README.md, USER_MANUAL.md）を最新の状態に更新します。

1. 最新のコードを読み込む
   - `Code.gs` を読み込む
   - `Dialog.html` を読み込む

2. 現在のドキュメントを読み込む
   - `README.md` を読み込む
   - `USER_MANUAL.md` を読み込む
   - `sheets_structure.md` を読み込む

3. 差分分析と更新提案
   - コードに追加された新しい関数や機能がないか確認する
   - 仕様変更（引数の変更、ロジックの変更）がないか確認する
   - ドキュメントとの乖離がある箇所を特定する

4. ドキュメントの更新（ユーザー承認後）
   - 必要な箇所を修正し、ドキュメントを更新する
