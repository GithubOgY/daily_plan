# 予定管理システム

あじさい園の業務予定・納品指示・ブログ記事を管理するシステムです。

## 📁 プロジェクト構成

```
practice/
├── 予定/                    # 予定関連ファイル
│   ├── YYYY-MM-DD_一日の予定.md          # 日次予定
│   ├── YYYY-MM-DD_企業名_納品指示書.md   # 納品指示書
│   ├── YYYY-MM_月間予定.md               # 月間予定
│   ├── 予定作成設定.json                 # 予定作成の設定
│   ├── 納品指示書作成設定.json           # 納品指示書作成の設定
│   └── 月間予定作成設定.json             # 月間予定作成の設定
├── ブログ/                  # ブログ記事
│   └── YYYY-MM-DD_タイトル_ブログ記事.html
└── README.md               # このファイル
```

## 🎯 機能

### 1. 一日の予定管理
- 送迎業務、利用者対応、納品業務などの日次スケジュールを管理
- Google Maps APIを使用した移動時間の自動計算
- 自宅、職場、納品先などの位置情報を設定ファイルで管理

### 2. 納品指示書作成
- 納品先別の納品指示書を自動生成
- 荷積み・納品作業・荷下ろしの時間を自動計算
- 移動時間の自動計算

### 3. 月間予定管理
- 月間スケジュールの一元管理
- 利用者の欠席・遅刻・早退などの特記事項を記録
- 職員の予定、会議、研修などの情報を管理

### 4. ブログ記事作成
- HTML形式でのブログ記事作成
- イベントレポート、活動報告などの記事を管理

## ⚙️ 設定ファイル

### 予定作成設定 (`予定/予定作成設定.json`)
- デフォルトの移動手段
- ラッシュアワー対応の時間バッファ
- よく使用する場所の住所とカテゴリ

### 納品指示書作成設定 (`予定/納品指示書作成設定.json`)
- 納品日によるフィルタリング設定
- デフォルトの出発時刻
- 荷積み・納品作業・荷下ろしの所要時間

### 月間予定作成設定 (`予定/月間予定作成設定.json`)
- 抽出するセクションの設定
- 除外するキーワード
- 出力フォーマットの設定

## 📝 ファイル命名規則

### 一日の予定
```
YYYY-MM-DD_一日の予定.md
例: 2025-11-26_一日の予定.md
```

### 納品指示書
```
YYYY-MM-DD_企業名_納品指示書.md
例: 2025-11-25_JPF_納品指示書.md
```

### 月間予定
```
YYYY-MM_月間予定.md
例: 2025-11_月間予定.md
```

### ブログ記事
```
YYYY-MM-DD_タイトル_ブログ記事.html
例: 2025-11-22_BCP机上訓練_ブログ記事.html
```

## 🚀 使用方法

1. **一日の予定を作成**
   - 日付、業務内容、移動先などの情報を提供
   - システムが自動的にスケジュールを生成

2. **納品指示書を作成**
   - 納品データ（日付、品目、数量など）を提供
   - システムが納品指示書を自動生成

3. **月間予定を作成**
   - 業務日誌などのデータを提供
   - システムが月間予定を抽出・整形

4. **ブログ記事を作成**
   - イベント情報や活動内容を提供
   - システムがHTML形式のブログ記事を生成

## 🔧 技術スタック

- **Markdown**: 予定ファイルの記述
- **JSON**: 設定ファイル
- **HTML**: ブログ記事
- **Google Maps Platform (MCP)**: 移動時間の計算

## 📋 注意事項

- 個人情報や機密情報が含まれるファイルは、GitHubにプッシュする前に確認してください
- 設定ファイルには個人情報が含まれる可能性があるため、必要に応じて `.gitignore` に追加してください
